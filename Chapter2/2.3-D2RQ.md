# 2.3 D2RQ

D2RQ平台一个能够把关系数据库当作虚拟的,只读的RDF图来访问的一个平台.它提供了基于RDF的访问模式,但不需要把数据以RDF的形式存储。

D2RQ特点

* 使用saprql查询非rdf数据库
* 把数据库的内容当作链接来访问
* 把数据库的内容以rdf的形式转化出来
* 使用pache jena api 访问非RDF数据库
